An e-commerce web application built with the **MERN stack** (MongoDB, Express.js, React, Node.js). NexStore provides a seamless shopping experience with secure authentication, product management, and a modern UI.

---

## 🚀 Features
- User authentication with JWT (signup/login)
- Product listing and details page
- Admin functionality for product management (CRUD)
- Responsive frontend built with React
- RESTful API backend with Express.js
- MongoDB database with Mongoose

---

## 🛠️ Tech Stack
- **Frontend:** React, Axios, Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB + Mongoose
- **Authentication:** JWT
- **Deployment:** (Netlify/Vercel for frontend, Render/Heroku for backend, MongoDB Atlas for database)

## ⚡ Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/DiviNexus/NexStore.git

2. Navigate to project folder:
   cd NexStore

3. Install dependencies:
   npm install

4. Run backend:
   cd backend
   npm start

5. Run frontend:
   cd frontend
   npm start
